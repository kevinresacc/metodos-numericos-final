Parámetros del Problema

Condiciones del problema:
  • Condición inicial: u(x,0) = sin(π·x)
  • Condiciones de frontera: u(0,t) = 0, u(L,t) = 0

Información del Método

Ecuación del Calor:
∂u/∂t = α ∂²u/∂x²
Condición de estabilidad (CFL):
r = α · Δt / (Δx)² ≤ 0.5

Si esta condición no se cumple, la solución puede volverse inestable.

Uil+1 = Uil + r (Ui+1l - 2Uil + Ui-1l)
